היכל התהילה

עבודות מצטיינות של סטודנטים ב-EconLabPPE — הכלכלה בפעולה.

הערהכיצד להיכנס להיכל התהילה?

השלימו אחד משלושת פרויקטי הגמר (שוק הדיור, מעקב אינפלציה, שוק העבודה) ברמה גבוהה, ושלחו Pull Request עם הקוד + דוח. הפרויקטים המצטיינים יוצגו כאן עם שם המגיש ותיאור הממצאים.

שוק הדיור הישראלי

Housing Market Analysis

ניתוח מגמות מחירי הדיור בישראל לאורך זמן, בחינת הקשר לריביות, לשכר ולביקוש. כולל מודל רגרסיה, ויזואליזציה אינטראקטיבית וחיזוי.

Python Pandas Plotly OLS API בנק ישראל

מעקב אינפלציה

Inflation Tracker

מעקב אחר אינפלציה בישראל ובעולם, השוואה בין מדינות, ניתוח מרכיבי CPI ובניית מודל ARIMA לחיזוי קצר-טווח.

R tidyverse ARIMA FRED API ggplot2

Labor Market Analysis

SQL Python Panel Data OECD API Power BI
