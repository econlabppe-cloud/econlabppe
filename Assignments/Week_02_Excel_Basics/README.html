חלק א – מה זה Excel ולמה כלכלנים צריכים אותו?

Excel הוא גיליון אלקטרוני – טבלה ענקית שאפשר לשים בה נתונים ולחשב עליהם.

מדוע Excel עדיין רלוונטי בעולם של Python ו-R?

  1. 90% ממקומות העבודה עדיין עובדים עם Excel (בנקים, משרד האוצר, חברות ייעוץ)
  2. קובלי מדיניות מקבלים דוחות ב-Excel, לא ב-Jupyter Notebook
  3. ניתוח ראשוני מהיר – אחרי שמורידים נתונים, Excel הוא הכי מהיר לבדיקה ראשונה
  4. תקשורת עם לא-כלכלנים – כולם יודעים לפתוח Excel

חלק ב – היכרות עם הממשק

פתח Excel / Google Sheets

אם אין לך Excel: השתמש ב-Google Sheets (חינמי לחלוטין, כנס ל-sheets.google.com).

הרכיבים הבסיסיים

┌──────────────────────────────────────────────┐
│  קובץ  ערוך  תצוגה  הוספה  פורמט  נוסחאות  │  ← שורת תפריטים
├──────────────────────────────────────────────┤
│  B2   │  =A1+B1                               │  ← תיבת שם | שורת נוסחה
├────┬──┴────┬──────┬──────┬──────┬─────────────┤
│    │  A    │  B   │  C   │  D   │             │  ← עמודות (אותיות)
├────┼───────┼──────┼──────┼──────┤
│  1 │       │      │      │      │             │  ← שורות (מספרים)
├────┼───────┼──────┼──────┼──────┤
│  2 │       │      │      │      │             │
├────┼───────┼──────┼──────┼──────┤

מושגים בסיסיים: - תא (Cell): הצומת של עמודה ושורה. A1 = עמודה A, שורה 1. - טווח (Range): קבוצת תאים. A1:A10 = כל התאים מ-A1 עד A10. - גיליון (Sheet): כרטיסייה אחת בקובץ (Sheet1, Sheet2…) - חוברת (Workbook): הקובץ עצמו (.xlsx)

חלק ג – הזנת נתונים ראשונה

משימה: בנה טבלת נתוני מאקרו ישראל

צור גיליון חדש בשם macro_data ובנה את הטבלה הבאה:

שנה צמיחת תמ”ג % אינפלציה % אבטלה % ריבית %
2018 4.2 0.8 4.0 0.1
2019 3.8 0.8 3.8 0.25
2020 -1.9 -0.6 4.3 0.1
2021 8.6 1.5 5.0 0.1
2022 6.5 5.1 3.7 3.25
2023 2.0 4.2 3.5 4.75
2024 1.1 3.5 4.2 4.5

איך להזין נתונים: 1. לחץ על תא A1 ← כתוב “שנה” ← לחץ Tab (עובר לתא הבא ימינה) 2. כתוב “צמיחת תמ”ג %” ← Tab → “אינפלציה %” ← Tab… וכן הלאה 3. Enter עובר לשורה הבאה 4. הזן את כל הנתונים

טיפ: אם הזנת שגיאה, לחץ Ctrl+Z לביטול.

חלק ד – עיצוב בסיסי

1. עיצוב כותרות

לחץ על שורה 1 (לחץ על המספר “1” משמאל) → כל השורה מסומנת.

  • מודגש: Ctrl+B
  • צבע רקע: לחץ על 🎨 (Fill Color) → בחר צבע כהה
  • צבע טקסט: לחץ על A עם קו מתחת → בחר לבן

2. עיצוב מספרים

לחץ על עמודה B (לחץ על האות “B”) → כל העמודה מסומנת. - לחץ Ctrl+1 (פתח עיצוב תא) - לשונית Number → Category: Number → Decimal places: 1 - לחץ OK

לעמודות C, D, E – אותו הדבר.

3. רוחב עמודות

כאשר מספרים מופיעים כ-###, העמודה צרה מדי: - לחץ פעמיים על גבול העמודה (בין A ל-B בכותרת) ← מתאים אוטומטית - או גרור את גבול העמודה לרוחב הרצוי

4. הפוך לטבלה (חשוב!)

  1. לחץ על כל תא בתוך הנתונים שלך
  2. Insert → Table (או Ctrl+T)
  3. ✅ “My table has headers” → OK

למה חשוב? טבלה מאפשרת סינון אוטומטי, ופורמולות שמתרחבות אוטומטית.

חלק ה – פורמולות ראשונות

כלל הזהב: כל פורמולה מתחילה ב-=

יצור גיליון חדש: analysis

בגיליון analysis, תחת כל כותרת, נחשב:

1. =SUM() – סכום

=SUM(B2:B8)

חשב את הסכום של כל ערכי צמיחת התמ”ג (2018-2024). (אמנם הסכום לא מאוד משמעותי כלכלית, אבל מתרגלים את הפונקציה)

2. =AVERAGE() – ממוצע

=AVERAGE(B2:B8)

שאלה כלכלית: מה הצמיחה הממוצעת של ישראל בשנים אלה?

3. =MAX() ו-=MIN() – ערך מקסימלי ומינימלי

=MAX(B2:B8)    → שנת הצמיחה הגבוהה ביותר (ערך)
=MIN(C2:C8)    → שנת האינפלציה הנמוכה ביותר (ערך)

4. =COUNT() ו-=COUNTA()

=COUNT(B2:B8)    → כמה תאים מכילים מספרים (= 7)
=COUNTA(A2:A8)   → כמה תאים לא ריקים (= 7)

5. =COUNTIF() – ספור לפי תנאי

=COUNTIF(C2:C8, ">2")

שאלה כלכלית: כמה שנים הייתה אינפלציה מעל 2%?

=COUNTIF(B2:B8, "<0")

שאלה כלכלית: כמה שנים הייתה ישראל במיתון (צמיחה שלילית)?

6. =IF() – תנאי

=IF(B2>2, "צמיחה גבוהה", "צמיחה נמוכה")

השתמש בזה ליצור עמודה חדשה בגיליון macro_data שמסווגת כל שנה.

חלק ו – Reference Absolute vs Relative (קריטי!)

זה אחד המושגים החשובים ביותר ב-Excel.

Reference יחסי (ברירת מחדל)

כאשר מעתיקים פורמולה – ה-reference זז יחסית:

A1 → A2 → A3... (כשמעתיקים למטה)
A1 → B1 → C1... (כשמעתיקים ימינה)

Reference מוחלט (עם $)

כאשר רוצים שתא מסוים לא יזוז:

$A$1    ← עמודה A קבועה, שורה 1 קבועה
$A1     ← עמודה A קבועה, שורה זזה
A$1     ← עמודה זזה, שורה 1 קבועה

לחץ F4 בזמן עריכת פורמולה לסבב בין האפשרויות.

דוגמה מעשית

נניח שיש לך שיעור ניכוי של 17% בתא G1, ורוצה לחשב מחיר אחרי ניכוי:

בתא E2: =D2*(1-$G$1)

כשמעתיקים את E2 למטה ל-E3, E4… – D2 יזוז ל-D3, D4…, אבל $G$1 ישאר G1.

חלק ז – מיון, סינון, חיפוש

מיון (Sort)

לחץ על עמודת האינפלציה → Data → Sort: - Sort by: אינפלציה % - Order: Largest to Smallest

שאלה: איזו שנה הייתה האינפלציה הגבוהה ביותר? (2022 – 5.1%)

סינון (Filter)

עם הטבלה שלך (אחרי Insert → Table), יופיעו חיצים בכותרות אוטומטית.

לחץ על חץ בעמודת “ריבית %” → Number Filters → Greater Than → כתוב 1

תראה רק שנים עם ריבית מעל 1% (2022, 2023, 2024).

כדי לבטל: לחץ על החץ → Clear Filter.

חיפוש (Ctrl+F)

לחץ Ctrl+F → כתוב ערך לחיפוש. מאוד שימושי בגיליונות גדולים.

חלק ח – גרף ראשון

בנה גרף קו של צמיחת התמ”ג

  1. סמן את עמודת “שנה” (A1:A8) ← Ctrl ← סמן את עמודת “צמיחת תמ”ג %” (B1:B8)
  2. Insert → Charts → Line → Line with Markers
  3. גרף בסיסי נוצר!

עיצוב מינימלי: - לחץ פעמיים על כותרת הגרף → כתוב: “צמיחת תמ”ג ישראל 2018-2024” - לחץ ימין על הגרף → Add Data Labels (מוסיף ערכים לכל נקודה) - לחץ על + (Chart Elements) → סמן Axis Titles → הוסף “שנה” לציר X ו-“%” לציר Y

משימות השבוע – תרגול עצמי

צור גיליון macro_data עם הנתונים הבאים (כולל 2015-2017 שתצטרך למצוא בבנק ישראל):

  • שנה (2015-2024)
  • צמיחת תמ”ג %
  • אינפלציה %
  • אבטלה %

מקור נתונים: boi.org.il → Statistics → Key Economic Indicators

משימה 2: גיליון ניתוח (40 נקודות)

צור גיליון analysis וחשב:

מדד פורמולה תוצאה
ממוצע צמיחה כל תקופה =AVERAGE ?
ממוצע אינפלציה כל תקופה =AVERAGE ?
שנת צמיחה מקסימלית =MAX ?
שנת מיתון =MIN ?
מספר שנים עם אינפלציה > 2% =COUNTIF ?
מספר שנים עם אינפלציה < 0% =COUNTIF ?
מספר שנים עם ריבית > 1% =COUNTIF ?

משימה 3: עמודת סיווג (30 נקודות)

הוסף לטבלה הראשית עמודה “סיווג מאקרו” שמשתמשת ב-IF: - אם צמיחה > 4% → “צמיחה גבוהה” - אם צמיחה > 0% → “צמיחה נמוכה” - אם צמיחה < 0% → “מיתון”

(רמז: השתמש ב-IF מקונן: =IF(B2>4, "צמיחה גבוהה", IF(B2>0, "צמיחה נמוכה", "מיתון")))

משימה 4: גרף (בונוס)

צור גרף קו שמציג שתי סדרות ביחד: צמיחת תמ”ג + ריבית.

(רמז: סמן שתי עמודות לא צמודות – Ctrl+Click)
