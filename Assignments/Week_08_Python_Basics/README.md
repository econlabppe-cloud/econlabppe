    """×× ×ª×—×ª ×¡×“×¨×ª × ×ª×•× ×™× ×•××—×–×™×¨×” ×¡×˜×˜×™×¡×˜×™×§×” ×‘×¡×™×¡×™×ª."""

    mean = sum(data_list) / n

    min_val = sorted_data[0]
