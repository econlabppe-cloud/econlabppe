## חלק ב – משימות

### 1. ניקוי ועיצוב (10 נקודות)
- הוסף כותרות ברורות לכל עמודה
- עצב כ-Table (Insert → Table)
- הוסף פורמט מספרי עם ספרה אחרת אחרי הנקודה
- צור גיליון נפרד `cleaned_data`

### 2. חישובים בסיסיים (20 נקודות)
בגיליון `analysis` חשב:
- ממוצע צמיחה 2010–2024: `=AVERAGE(...)`
- שנת הצמיחה הגבוהה ביותר: `=MAX(...)` + `=INDEX/MATCH`
- מספר שנים עם אינפלציה > 3%: `=COUNTIF(...)`
- מתאם בין צמיחה לאינפלציה: `=CORREL(...)`

### 3. Pivot Table (30 נקודות)
צור Pivot Table שמציג:
- ממוצע אינפלציה לפי עשור (2010–2019 vs 2020–2024)
- שנים עם ריבית > ממוצע: True/False בעמודה חדשה
- השתמש ב-Slicer לסינון לפי טווח שנים

### 4. גרפים (40 נקודות)
**גרף 1 – סדרה עתית:**
- ציר X: שנה, ציר Y: צמיחת תמ"ג
- הוסף קו מגמה (Trendline)
- עצב כך שאפשר להציג בפרזנטציה

**גרף 2 – Scatter Plot:**
- ציר X: אינפלציה, ציר Y: ריבית בנק ישראל
- כל נקודה = שנה אחת
- הוסף תווית לנקודות שנת 2022-2024 (עידן הריבית הגבוהה)

<details>
<summary><b>שאלה 1: מה ההבדל בין הפניה יחסית (A1) להפניה מוחלטת ($A$1)?</b> 👁️ לחץ לתשובה</summary>
<br>
הפניה יחסית תשתנה אוטומטית כאשר נגרור את הנוסחה לתאים אחרים. הפניה מוחלטת (עם סימני הדולר) "נועלת" את התא, כך שגם אם נגרור את הנוסחה, היא תמיד תצביע לאותו תא ספציפי (שימושי מאוד לקיבוע משתנים כמו שער חליפין או מע"מ).
</details>

<details>
<summary><b>שאלה 2: מהי החולשה המרכזית של פונקציית VLOOKUP?</b> 👁️ לחץ לתשובה</summary>
<br>
VLOOKUP מחפשת את הערך <b>אך ורק בעמודה הימנית ביותר</b> (או השמאלית ביותר באנגלית) של טווח הנתונים. אם הערך שאנחנו רוצים להחזיר נמצא מימין לעמודת החיפוש, VLOOKUP לא תעבוד ונצטרך לשנות את מבנה הטבלה או להשתמש בפונקציות מתקדמות יותר.
</details>
