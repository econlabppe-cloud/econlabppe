# משימות שבועיות – EconLab PPE

## איך זה עובד?

כל שבוע תפרסם בקבוצת WhatsApp:
1. סרטון/מדריך קצר על הנושא
2. המשימה המפורטת (בקובץ README.md בתיקיית השבוע)

**הגשה:** העלה את הפתרון שלך לתיקייה האישית שלך ב-`Members/YourName/Week_XX/`

## הגשת משימה – איך?

```bash
# 1. ודא שאתה על ה-branch הנכון
git checkout main

# 2. צור תיקייה לשבוע הנוכחי
mkdir Members/YourName/Week_01_Setup

# 3. שים בה את הפתרון שלך
# (קובץ .py / .R / .ipynb / צילום מסך / README.md)

# 4. Commit + Push
git add Members/YourName/
git commit -m "feat: Week 1 setup - YourName"
git push
```
